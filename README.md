TravelLite is a web-based ticket booking system developed using Oracle APEX.

This application was built and deployed locally using Oracle REST Data Services (ORDS) 
running on a localhost environment instead of the Oracle APEX cloud workspace.

The system allows users to:
• Register and log in
• View available services
• Book tickets
• Generate invoices
• Print booking invoices

Key Features:
- User Registration & Authentication
- Service Listing (Flights / Hotels)
- Ticket Booking Workflow
- Invoice Generation
- Print-friendly invoice layout
- Responsive UI for desktop and mobile

Technologies Used:
- Oracle APEX
- Oracle REST Data Services (ORDS)
- Oracle Database
- SQL / PLSQL
- HTML / CSS
